# 08 — Experiments

A verbatim fork of **`07-perf-visible-godrays`** kept as a sandbox to try new
ideas without disturbing v07. Same stack (Vite + vanilla JS + Three.js +
`postprocessing` + `three-good-godrays` + `lil-gui`), same auto/walk mode picker,
graphics tiers, visible godrays, FluffyGrass-style grass, and the live tuning GUI.

Run: `npm install && npm run dev` (port 5180).
