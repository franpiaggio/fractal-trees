# 16 — PS1 (explorations)

A fork of **`15-ps1`** to keep exploring the PlayStation-1 look. Same toolkit
(`src/ps1.js`: resolution-tied vertex jitter, nearest+mipmap textures, 15-bit
colour + Bayer dither; modern post stripped; LINEAR tone-map with lowered
exposure). Audio is OFF by default. Run: `npm install && npm run dev` (port 5188).

Live **PS1** GUI folder (desktop): internal resolution, vertex jitter, colour
levels, dither.

Scratchpad for further PS1 ideas (affine warp, banding, palette, fog, etc.).
